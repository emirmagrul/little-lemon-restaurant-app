## ğŸ§  Technical Decisions

### ğŸ”¹ Why useReducer?
useReducer was chosen to make state transitions more predictable, especially for dynamically updating available reservation times.

### ğŸ”¹ Why Vite?
- Faster development server
- Optimized production builds
- Native ES module support

### ğŸ”¹ Routing Strategy
React Router was used to build the SPA architecture.  
A Vercel rewrite rule was added to prevent 404 errors on page refresh.
